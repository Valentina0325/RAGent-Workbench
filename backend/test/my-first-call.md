  “你是怎么解决第一个技术卡点的”

“当时我第一次调用大模型API，在安装 `requests` 库时一直报网络超时（`ConnectionResetError`）。我意识到可能是默认的 PyPI 官方源在国外，国内访问不稳定。于是我去搜索了国内镜像源，换成清华源后顺利安装。事后我还把 `pip install （库名）-i https://pypi.tuna.tsinghua.edu.cn/simple` 这个命令记在了笔记里，后面所有环境搭建都更快了。这个经历让我学会了：遇到网络问题先考虑镜像源，而不是盲目重试。”
